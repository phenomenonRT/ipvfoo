# IPvFoo Collector Server

Локальный сервер на **Python 3** (без сторонних зависимостей) для сохранения пар домен+IP, которые фиксирует расширение IPvFoo.

## Требования

- Python 3.8+ (только стандартная библиотека, ничего устанавливать не нужно)

---

## Запуск на Windows

### Обычный запуск (с окном консоли)
```
python server.py
```

### Скрытый запуск (без окна) — рекомендуется
Дважды кликните по файлу:
```
start_server.vbs
```
Сервер запустится в фоне, окно не появится.

### Открыть веб-интерфейс
```
open_ui.bat
```

### Остановить сервер
```
stop_server.bat
```

---

## Автозапуск при входе в Windows

Добавить в автозапуск (без прав администратора):
```powershell
powershell -ExecutionPolicy Bypass -File install_autostart.ps1
```

Удалить из автозапуска:
```powershell
powershell -ExecutionPolicy Bypass -File install_autostart.ps1 -Uninstall
```

---

## Запуск на Linux / macOS
```bash
python3 server.py
```

---

## Веб-интерфейс

Откройте в браузере: **http://127.0.0.1:3456**

- Уникальные пары домен+IP
- История всех визитов
- Поиск по домену или IP
- Статистика

## API

| Метод | URL | Описание |
|-------|-----|----------|
| `POST` | `/api/collect` | Принять записи от расширения |
| `GET`  | `/api/entries` | Уникальные пары домен+IP |
| `GET`  | `/api/visits`  | История всех визитов |
| `GET`  | `/api/stats`   | Краткая статистика |

## Настройка в расширении

1. Запустите сервер (`start_server.vbs`)
2. Откройте настройки расширения IPvFoo (иконка → Options)
3. В секции **«Сервер сбора данных»** включите переключатель
4. Нажмите **«Проверить»** — должно появиться подтверждение связи

## Данные

База данных SQLite сохраняется в `server/ipvfoo.db` автоматически.
